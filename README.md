# Khaled Bouarbi – Portfolio

A modern single-page portfolio built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**, with Framer Motion and tsparticles for animations.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Scripts

| Command        | Description              |
|----------------|--------------------------|
| `npm run dev`  | Start development server |
| `npm run build`| Production build         |
| `npm run preview` | Preview production build |

## Project structure

- **`public/images/`** – Static images (profile, projects, certificates).
- **`src/components/layout/`** – Navbar, Footer, ParticleBackground.
- **`src/components/ui/`** – shadcn/ui components.
- **`src/sections/`** – Page sections (Home, Skills, Projects, etc.).
- **`src/lib/`** – Utilities and animation helpers.
- **`docs/`** – Technical notes and spec.

Path alias `@/` points to `src/` for imports.

## Tech stack

React 19 · TypeScript · Vite · Tailwind CSS · Framer Motion · Lucide React · tsparticles · shadcn/ui
